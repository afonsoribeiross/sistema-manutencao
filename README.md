# Sistema de Manutenção Industrial

## Demo
https://sistema-manutencao-kappa.vercel.app

Aplicação web desenvolvida com React, TypeScript e Vite para controle e visualização de máquinas industriais, permitindo cadastro, filtragem por status e acompanhamento de temperatura.

## Tecnologias
- React
- TypeScript
- Vite
- Supabase (PostgreSQL)
- CSS
- Git/GitHub

## Funcionalidades
- Cadastro de máquinas
- Classificação por status: Normal, Alerta e Falha
- Filtro por status
- Exibição de temperatura
- Interface responsiva em tema escuro

## Próximas melhorias
- Autenticação de usuários
- Histórico de manutenções
- Dashboard com indicadores
- Testes E2E com Playwright
- CI/CD com GitHub Actions

## Objetivo do projeto
Este projeto faz parte da evolução do meu portfólio técnico, conectando conhecimentos de Engenharia Mecânica, manutenção industrial e desenvolvimento web moderno.
